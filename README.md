# Pizza-Sales

This project showcases a fully interactive Pizza Sales Dashboard built using Microsoft Excel, designed to analyze sales performance, customer behavior, and product trends for a pizza store. The dashboard converts raw sales data into meaningful insights through dynamic charts, pivot tables, and automated KPIs.
